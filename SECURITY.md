# Security policy

## Supported versions

This repository is a **profile README** project (documentation and static assets). Treat the default branch as the supported surface.

## Reporting a vulnerability

If you believe you have found a security vulnerability that affects this repository or linked automation, please report it responsibly:

- Prefer **GitHub private vulnerability reporting** for this repository (if enabled under **Security**).
- Otherwise, contact the repository owner via the email listed on their GitHub profile / README.

Please do **not** open a public issue for undisclosed security vulnerabilities.
